# Huddle Landing Page

Responsive landing page created as a frontend practice project.

## Live Demo

https://svitlana-melenchenko.github.io/Huddle/

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Responsive Design
- SVG Sprite
- Git & GitHub

## Features

- Semantic HTML markup
- Fully responsive design
- Mobile and desktop layouts
- Flexible layout using Flexbox
- Hover and focus effects
- Email subscription form with validation
- Optimized images (WebP, 1x/2x)
- SVG sprite icons
